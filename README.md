# Basic Messenger

This is a basic messenger app.

# Install

`$ yarn install`
`$ yarn start`

# Design

<App>
  <ChatWindow>
    <MessageList>

    </MessageList>
    <MessageInput />
  </ChatWindow>
  <ContactList>
    <Contact>
      <Avatar />
      <Name />
      <Status />
    </Contact>
  </ContactList>
</App>
